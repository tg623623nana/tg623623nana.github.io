Distributed Multi-Agents Trajectory Planning With Games Using ILQR

This project involved implementing potential game theory and an iterative Linear Quadratic Regulator (iLQR) algorithm to address distributed trajectory planning in an interactive model.

Deadlock Occurrence in Narrow-Way Scenarios#

In multi-agent narrow-way scenarios, failure to establish communication among agents can lead to deadlocks.

Resolving the deadlock#

To tackle the deadlock problem, we introduced two crucial concepts: potential game theory and the safety distance property.

We formulated the trajectory optimization problem as a potential game, where each agent imagines cooperation from others and predicts their trajectories. By solving this problem, we obtained a feasible solution that achieved Nash equilibrium. This means that the solution represents the best strategy for all agents in a non-cooperative situation. Furthermore, by assigning different safety distances to each agent, we could generate diverse behaviors in the interactive model.
